## Tips for Best Results

1. **Midjourney**: Add `--ar 16:9` for wide images, `--ar 1:1` for squares
2. **DALL-E 3**: Specify dimensions in the prompt clearly
3. **Consistency**: Generate hero images in same session for consistent style
4. **Upscaling**: Use built-in upscalers or tools like Topaz for final output
5. **No text/watermarks**: Avoid prompts that might add text overlays

Once images are generated, let me know and I'll update the HTML to use them!
